# Project-Devsign-Board 소개
## 개요
<ul>
<li> 게시글 및 파일 업로드와 실시간 채팅이 가능한 회원제 게시판입니다.
<li> 모바일 환경을 고려하여 반응형 레이아웃을 적용했습니다.
</ul>

## 기술 스택
<ul>
<li> Apache
<li> PHP
<li> MariaDB
<li> Express.js
<li> Redis
</ul>

## 라이브러리
<ul>
<li> bootstrap
<li> JQuery
<li> Datatables
<li> PHPMailer
<li> Soekct.io
</ul>

## 목록
<ul>
<li> 회원가입 및 이메일 인증
<li> 메일을 통한 회원 정보 찾기 및 비밀번호 초기화
<li> 게시글 업로드, 수정, 삭제
<li> 파일 업로드, 수정, 삭제
<li> 댓글 추가, 수정, 삭제
<li> 세션 공유
<li> 소켓 통신을 이용한 실시간 채팅

</ul>
